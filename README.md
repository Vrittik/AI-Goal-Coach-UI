# AI Goal Coach UI

This is the frontend for the **AI Goal Coach** application.

The UI allows users to enter a vague goal and receive a refined SMART goal with measurable key results generated by the backend AI service.

---

## Tech Stack

- React.js
- Vite
- Axios

---

## Features

- Enter a goal in a text input
- Refine the goal using the AI backend
- Display the refined goal and key results
- Save goals
- View previously saved goals

---

## Setup

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

The application will run on:

```
http://localhost:5173
```

---

## Backend Requirement

The backend API must be running locally for the UI to work.

Default backend URL used:

```
http://localhost:5249/api/goals
```

---

## Notes

The UI communicates with the backend API to:

- refine goals using the AI model
- save refined goals
- fetch saved goals
